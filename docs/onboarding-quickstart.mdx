---
title: "Quick Start"
description: "Set up Dosu on a GitHub repository"
---
<Steps>
  <Step title="Create an Account">
    Head to [app.dosu.dev](app.dosu.dev) and authenticate with your GitHub account.
  </Step>
  <Step title="Install the GitHub App">
    From the [Dosu Github App install page](https://github.com/apps/dosubot/installations/select_target), install Dosu on the repositories you'd like.
  </Step>
  <Step title="Create a Workspace">
    In the app: click ["Create Workspace"](https://app.dosu.dev/workspaces/create), select a repository, and deploy.
  </Step>
</Steps>
