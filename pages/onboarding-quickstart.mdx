---
title: "Quick Start"
description: "Set up a Dosu workspace on a GitHub repository"
---
<Steps>
  <Step title="Create an Account">
    Head to [app.dosu.dev](https://app.dosu.dev) and authenticate with your GitHub account.
  </Step>
  <Step title="Install the GitHub App">
    From the [Dosu Github App install page](https://github.com/apps/dosubot/installations/select_target), install Dosu on the repositories you'd like.
  </Step>
  <Step title="Create a Workspace">
    Click ["Create Workspace"](https://app.dosu.dev/workspaces/create) in the app, select a repository, and deploy.
  </Step>
</Steps>
